## 🧠 Problem Statement
This project analyzes global air quality data to understand pollution trends and classify air quality levels.  

The main objective is to:
- Predict AQI categories using machine learning models  
- Identify pollution patterns using clustering techniques  
- Reduce dimensionality using PCA for better visualization  

---

- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Google Colab  

---

## 🚀 How to Run the Project
1. Open **Google Colab** or **Jupyter Notebook**  
2. Upload the dataset file  
3. Open `AQI_Data_Science_Project.ipynb`  
4. Run all cells from top to bottom  
5. Ensure all outputs and visualizations are displayed  

---

## 📌 Summary of Main Findings
- Naive Bayes performed better than KNN for AQI classification  
- Best KNN accuracy: **0.25**  
- Naive Bayes accuracy: **0.3866**  
- K-Means successfully grouped data into **3 pollution levels**  
- PCA helped visualize data in **2D space**  
- PM2.5 showed a strong relationship with AQI
